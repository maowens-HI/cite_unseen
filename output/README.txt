================================================================================
OUTPUT DIRECTORY
================================================================================

PURPOSE
-------
This folder contains all analysis outputs including figures, tables, and
reports generated by the analysis pipeline.

SUBDIRECTORIES
--------------

/figures/
  - PNG/PDF plots for publication
  - Diagnostic visualizations
  - Exploratory data analysis charts

/tables/
  - Regression results (CSV, LaTeX)
  - Summary statistics
  - Match score distributions

/reports/
  - Analysis summaries (Markdown, HTML)
  - Validation reports
  - Sensitivity analysis results

NAMING CONVENTIONS
------------------
Figures:
  fig_[number]_[description].[ext]
  Example: fig_01_match_score_distribution.png
           fig_02_did_estimates.pdf

Tables:
  tab_[number]_[description].[ext]
  Example: tab_01_summary_statistics.csv
           tab_02_regression_results.tex

Reports:
  report_[date]_[description].[ext]
  Example: report_20241201_preliminary_results.html
           report_20241215_sensitivity_analysis.md

VERSION CONTROL
---------------
- Commit key outputs (small figures, tables) to git
- Do NOT commit large files or frequently regenerated outputs
- Use descriptive commit messages for output changes
- Tag releases with version numbers

REPRODUCIBILITY
---------------
Every output should be reproducible from:
  1. Raw data in /data/raw/
  2. Scripts in /src/
  3. Configuration in /src/00_config.R

Document the exact command to regenerate each output:

  # Generate Figure 1
  Rscript src/05_score_analysis.R --output fig_01

  # Generate all tables
  Rscript src/06_causal_estimation.R --tables
